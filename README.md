# Shinobi Fighter ⚔️

Jeu de combat mobile en HTML5/JavaScript.

## Comment jouer

Ouvre `index.html` dans un navigateur (ou déploie sur Replit / GitHub Pages).

### Contrôles clavier
| Touche | Action |
|---|---|
| ← / → | Déplacer |
| ↑ / Espace | Sauter |
| Z | Attaque normale |
| X | Attaque spéciale |

### Contrôles tactiles (mobile)
Utilise les boutons à l'écran (◀ ▶ Saut Attaque Spécial).

## Structure
```
index.html   – page principale
style.css    – styles & mise en page
game.js      – logique du jeu (moteur, IA, combattants)
```
